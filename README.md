# Vicolor  - Generador de paletas

Aplicación web que permite generar paletas de colores aleatorias en diferentes tamaños y formatos.
Está diseñada para cualquier persona que necesite colores en formatos específicos como RGB, HSL o HEX.

## Caracteristicas 
1. Generación de colores aleatorios
2. Selección del formato de color:
- RGB
- HSL
- HEX
3. Selección del tamaño de la paleta:
- 6 colores
- 8 colores
- 9 colores
4. Animación de aparición de los colores
5. Interfaz simple e intuitiva

## Tecnologías usadas
- HTML
- CSS
- JAVASCRIPT

## ¿Como usar?
1. Abrir index.html en el navegador 
2. Clonar el repositorio 
3. seleccionar:
- Tipo de color 
- Tamaño de pañeta 
4. Presionar boton Generar Paleta
